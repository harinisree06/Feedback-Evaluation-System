# Feedback-Evaluation-System
Feedback Management System is an automated platform that collects student feedback through Jotform, stores responses in Airtable, and displays them in a Bubble dashboard. The system integrates multiple tools to create a seamless workflow for collecting, storing, and visualizing feedback in real time.

## Project Overview

Students submit feedback through an online form. The responses are automatically stored in a structured database and displayed in a dashboard where administrators can view the feedback submissions.

## System Workflow

Jotform Form Submission  
↓  
Make Automation Scenario  
↓  
Airtable Database  
↓  
Bubble API Connector  
↓  
Bubble Dashboard (Repeating Group)

## Tools Used

| Tool | Purpose |
|-----|--------|
| Jotform | Collect student feedback |
| Airtable | Store feedback records |
| Make (Integromat) | Automate data transfer |
| Bubble | Build the dashboard interface |
| Bubble API Connector | Fetch Airtable data |
| Repeating Group | Display feedback records dynamically |

## Features

- Online feedback submission form
- Automated data storage
- Real-time dashboard visualization
- Integrated workflow across platforms
- Simple and interactive user interface

## Challenges Faced

- Understanding Airtable API authentication
- Configuring API headers in Bubble
- Debugging workflow conditions
- Learning integration between multiple platforms
